Casos de uso para WPS (Website Payments Standard)
O Website Payments Standard (WPS) é uma integração legada do PayPal que permite aceitar pagamentos no seu site usando botões e páginas de pagamento hospedados pelo PayPal. Com o WPS, você pode criar botões de pagamento através do painel de controle da sua conta PayPal sem precisar escrever código. Quando os clientes clicam nesses botões, eles são direcionados para o PayPal para concluir a compra.

O PayPal agora oferece soluções modernas com recursos aprimorados, melhores experiências em dispositivos móveis e mais opções de personalização para ajudar você a expandir seus negócios.

Identificação de casos de uso
Para encontrar o caminho de atualização ideal para o seu negócio, primeiro identifique qual caso de uso do WPS corresponde à sua configuração atual. Analise as descrições abaixo para determinar qual delas melhor descreve como você aceita pagamentos hoje.

1. Preço definido pelo cliente
Você está usando este caso de uso se os clientes puderem inserir o valor do pagamento no seu site. Isso é comumente utilizado para doações, gorjetas, serviços profissionais ou qualquer situação em que o valor do pagamento varie de cliente para cliente.

Recomendamos: Atualize para os botões de pagamento do PayPal.


Principais características
Campo de valor personalizado
Sem preço fixo
Fatura/nota opcional

2. Botão de compra simples
Você está usando esse caso de uso se vende produtos ou serviços individuais a preços fixos usando botões do PayPal. Cada botão está vinculado a um único item com um preço definido, e os clientes compram um item por vez.

Recomendamos: Atualize para os botões de pagamento do PayPal.

Principais características
Produtos com preço fixo
Normalmente de 1 a 3 itens
Variantes opcionais (tamanho, quantidade, cor, etc.)

3. Carrinho de compras hospedado pelo PayPal
Você utiliza esse caso de uso quando os clientes podem adicionar vários produtos ao carrinho antes de finalizar a compra. O PayPal hospeda e gerencia toda a experiência do carrinho de compras, incluindo a página do carrinho e o fluxo de finalização da compra.

Recomendamos: Atualize para os botões do carrinho de compras do PayPal.

Principais características
Vários produtos (geralmente menos de 20)
Gestão básica de estoque
Carrinho hospedado pelo PayPal
